# 位置-任务-成功标准 与 组合原则

## 核心定位

这是写作时的**自检表**——写到某一节时，回头查表：这一节的"位置-任务-成功标准"是什么？我做到了吗？

成功标准用来衡量用户是否读透了、写到位了，而非衡量 AI 的输出质量。

## 学术述评类论文：位置-任务-成功标准

### 第一节位置

| 维度 | 内容 |
|------|------|
| **任务（功能标注）** | 系统梳理该维度的已有研究 |
| **成功标准** | ① 读者能说出该维度的核心流派/人物/主张 |
|  | ② 为后文提供分析基础 |
|  | ③ 该维度的内容能被第四节横向透视所调用 |

### 第二节位置

| 维度 | 内容 |
|------|------|
| **任务（功能标注）** | 第二重维度展开，保持论述的系统性 |
| **成功标准** | ① 与第一节构成并列或递进关系 |
|  | ② 每个维度都有代表人物和核心主张 |

### 第三节位置

| 维度 | 内容 |
|------|------|
| **任务（功能标注）** | 第三重维度展开，形成三维分析格局 |
| **成功标准** | ① 与第一、二节构成完整的分析体系 |
|  | ② 三个维度之间不重复、不遗漏 |

### 第四节位置（核心论点所在）

| 维度 | 内容 |
|------|------|
| **任务（功能标注）** | 横向透视前三节 |
| **成功标准** | ① 读者能发现前三节之间的内在联系 |
|  | ② 提出共性局限/共识/趋势 |
|  | ③ 这些共性分析不是显而易见的，而是需要深度洞察才能发现 |

### 结论位置

| 维度 | 内容 |
|------|------|
| **任务（功能标注）** | 升华研究意义 |
| **成功标准** | ① 意义必须有具体三点，不能空泛 |
|  | ② 意义的第一点应直接回应引言的"本文任务" |

## 使用方式

### 写作时（查表）

```
写到某一节时：
1. 查表：这一节的"位置-任务-成功标准"是什么？
2. 自检：我做到了吗？
3. 如未达标 → 调整内容
```

### 写完后（覆盖自检）

```
全文写完后：
1. 逐节对照成功标准
2. 标记未达标的部分
3. 修订至每段每句 100% 覆盖
```

## 组合原则：四维一体

| 维度 | 说明 | 单独使用的问题 |
|------|------|--------------|
| **谋篇模板** | 这类论文的整体结构框架 | 结构对了但可能任务错位 |
| **位置-任务-成功标准** | 每个部分做什么、做到什么程度 | 单节合格但整体可能不连贯 |
| **常见错误** | 这类论文写作时容易犯的错误 | 无法从结构层面规避 |
| **评判标准** | 好坏从哪几个维度评判 | 写完无法自评 |

**"组合"含义**：这四个维度互为约束、缺一不可，必须同时抽取、同时使用。

## 章节功能标注 vs 位置-任务-成功标准

| | 功能标注 | 成功标准 |
|---|---------|---------|
| **性质** | 描述性 | 评估性 |
| **回答** | "这一节做什么" | "做到什么程度算做好" |
| **关系** | 是成功标准的第一栏"任务" | 是功能标注的完成度验证 |

## 同一类 vs 不同类论文

| | 同一类论文 | 不同类论文 |
|---|----------|----------|
| **章节功能标注结构** | 相同 | 不同 |
| **成功标准** | 相同 | 不同 |
